# Missing Person Identification Using Computer Vision

This project uses Python and face recognition to identify missing persons through webcam detection.

## 📂 Project Features
- Real-time face detection using `face_recognition` and OpenCV
- Database connection using `pymysql` (MySQL)
- Tkinter GUI to display results
- "Face Not Matched" alert for unknown faces

## 🛠️ Technologies Used
- Python 3.8
- face_recognition
- OpenCV
- MySQL
- Tkinter

## ▶️ How to Run
1. Install dependencies from `requirements.txt` (if available)
2. Start MySQL server and ensure the `Student` database is running
3. Run the script: `python face.py`

## 📸 Images
Put training images inside the `imagesattendence/` folder
